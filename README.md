# FEAT Condensed Matter

Code for FEAT part in the paper "Assessing generative modeling approaches for free energy estimates in condensed
matter"

### Train FEAT

```
bash run_mw_all.sh
bash run_lj_fcc_all.sh
bash run_lj_hcp_all.sh
```

### Test FEAT
```
bash test_mw.sh
bash test_lj_fcc.sh
bash test_lj_hcp.sh
```
